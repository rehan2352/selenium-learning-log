This project is a Selenium WebDriver example that demonstrates how to handle dropdown elements using the Select class.

What this project does

Opens Chrome browser and navigates to a demo site.

Locates a dropdown element.

Selects options using:

selectByVisibleText()

selectByValue()

selectByIndex()

Retrieves and prints all dropdown options.

Requirements

Java JDK 8 or higher

Selenium WebDriver

Chrome Browser

ChromeDriver installed and configured in PATH

How to run

Open the project in your IDE.

Add Selenium WebDriver libraries.

Run the SelectDropDown.java file.

Output

Chrome browser will open the test site.

Dropdown selection will be performed.

Console will show:

Total number of dropdown options.

Names of all available options.
