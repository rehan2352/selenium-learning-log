# ⏱️ Wait Methods in Selenium WebDriver

This Java program demonstrates the use of **Wait Methods** in Selenium WebDriver:

- ✅ **Explicit Wait** using `WebDriverWait` and `ExpectedConditions`
- 🔁 (Optional) **Implicit Wait** using `driver.manage().timeouts().implicitlyWait()`

Waits are used to handle dynamic elements or delays in loading elements in real-time web applications.

---

## 🚀 Technologies Used

- Java
- Selenium WebDriver
- ChromeDriver

---

## 🧪 Script Overview

This program:

1. Launches Chrome browser.
2. Navigates to the [Automation Testing Demo Site](https://demo.automationtesting.in/Register.html).
3. Maximizes the browser window.
4. Applies **Explicit Wait** to wait until the password confirmation field is visible.
5. Sends text `"rehan@hbd"` to the field.

> Note: Implicit Wait code is present but commented out. You can enable it by uncommenting.
